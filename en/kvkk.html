Information Text on the Protection of Personal Data

As a sole proprietorship operating as a data controller, in accordance with the Personal Data Protection Law No. 6698 ("KVKK"), we would like to inform you about your personal data processed within the scope of the Vaktim mobile application.

Communication: support@vaktim.app

1. Processed Personal Data

The following personal data can be processed within the scope of the application:

  • Email address
  • Username and profile information
  • avatar image
  • Premium subscription status information
  • Notification permission status
  • Worship and progress data (prayer tracking, dhikr numbers, prayer reading statistics, Quran progress information, etc.)
  • Daily quest completion information
  • Favorite content records
  • Spiritual guide chat contents (user-generated message history)
  • Usage counter information for the purpose of managing service usage limits (e.g. number of chat usages)

The app does not collect advertising, analytics or behavioral tracking data.

2. Purposes of Processing Personal Data

Your personal data;

  • Creating an account and carrying out identity verification processes,
  • Managing profile information,
  • Storing and displaying worship and progress data,
  • Providing cloud synchronization with the user's explicit consent,
  • Carrying out premium subscription transactions,
  • Checking service usage limits,
  • Answering user support requests

is processed for its purposes.

3. Cloud Sync and Explicit Consent

The cloud sync feature is turned off by default.

This feature is only activated if the user gives explicit in-app consent.

When cloud syncing is active:

  • Worship and progress data,
  • favorites,
  • tasks,
  • chat history,
  • Counter and summary data

It can be associated with the user's account and stored in the server environment.

When synchronization is turned off, new data is not sent to the server. The user can also delete his data on the server with the "Delete My Data in the Cloud" option.

4. Data Transfer

4.1 Supabase (Germany – Frankfurt / European Union)

Account and synchronization data are stored in the Supabase infrastructure located within the borders of the European Union.

This transfer;

  • Account management
  • data backup
  • Cloud synchronization

It is done for the purposes.

4.2 RevenueCat

Premium subscription transactions are carried out through the RevenueCat infrastructure.

Technical information regarding subscription status may be shared with this service provider.

Your data will never be sold to third parties for advertising or marketing purposes.

5. Location and Notification Usage

Location information is used only for calculating prayer times on the device and is not transferred to the server.

Notifications are scheduled only on the device (local notifications). A server-based push notification system is not used.

6. Biometric Verification

Face ID or similar biometric authentication is performed only through the device's built-in security system.

Biometric data is not processed and stored by the application or server.

7. Storage Period

Your personal data:

  • As long as your account is active,
  • To be deleted from the systems within a reasonable time upon your account deletion or data deletion request.

is stored.

Data is not automatically deleted when cloud syncing is turned off; It is deleted if the user requests it.

8. Your Rights Under KVKK

In accordance with Article 11 of KVKK;

  • Learning whether your personal data is being processed or not,
  • If it has been processed, request information about it,
  • Learning whether it is used for its intended purpose or not,
  • Learning the third parties to whom it is transferred domestically or abroad,
  • Requesting correction if it is processed incompletely or incorrectly,
  • Requesting its deletion or destruction,
  • Requesting notification to third parties to whom the transfer is made,
  • Claiming compensation if you suffer damage as a result of processing

You have the rights.
